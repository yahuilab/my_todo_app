# 🚀 My-AI-Todo-App

这是我作为大二 CS 学生，使用 **Cursor AI** 辅助开发的首个全栈 Web 小程序。

---

### ✨ 功能亮点
* **任务管理**：支持添加、标记完成及一键删除。
* **数据存储**：集成 SQLite 数据库。
* **响应式 UI**：采用 Bootstrap 5 构建。

### 🛠️ 技术栈
* **后端**: Python (Flask)
* **前端**: HTML5 + Bootstrap 5
* **工具**: Cursor AI + GitHub
